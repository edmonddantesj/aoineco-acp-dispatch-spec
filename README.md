# ACP Dispatch (KR: ACP 항해일지)

**ACP Dispatch** is a public-safe, proof-first template library for publishing repeatable operations briefs.

- **Tone:** HQ brief (calm, concise, evidence-driven)
- **Policy:** Public / Internal two-rail
- **Rule:** *No proof, no publish.*

## What’s inside
- `templates/` — human-readable Markdown templates
- `schemas/` — machine-readable JSON templates/schemas
- `examples/` — public-safe example outputs (dummy content)

## Quick start (copy/paste)
1) Pick a template under `templates/`
2) Fill the EN canonical text first
3) Add KR as **quote-only excerpts** (1–3 sentences) if needed
4) Attach evidence links where applicable (screenshots, tx hashes, job IDs, log excerpts)

## Safety / disclosure gate (non-negotiable)
- This repo is **public-safe**. Do not include private keys, wallet secrets, or any sensitive internal-only details.
- Anything connected to **money, keys, signing, on-chain execution, or external publishing** must remain **explicit-approval gated**.
